## Review result

The original inventory covered the 9 generated plots, 8 fitted/diagnostic models, and the core fit/CV/deployment metrics. This update adds the report sections that were listed as open questions or were only implicit in the report generator:

- reference-chain verification table fields,
- event counts and event summary fields,
- accepted hold dataset fields,
- holdout prediction fields,
- creep/zero-return summary fields,
- dynamic validation summary fields,
- previous-calibration comparison procedure,
- firmware HX711-style scale/offset export,
- extended `calibration_dataset.csv` concept groups from the report/workflow documentation.

Status legend: `Y` = covered by the extended static reference and by at least one source in the reference inventory. `D` was removed because this pass treats every displayed or documented concept as worthy of static explanation.
