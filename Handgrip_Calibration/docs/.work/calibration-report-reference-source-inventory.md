## Concept-to-source coverage matrix

| Concept group                                                    | Primary source IDs                  | Notes                                                                                                                                         |
| ---------------------------------------------------------------- | ----------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Force/load-cell calibration workflow                             | R02, R17, R18                       | Covers fitting a sensor response to known/reference loads and using the model for future measurements.                                        |
| Residuals and residual plots                                     | R01, R15                            | Defines residuals and explains graphical residual analysis.                                                                                   |
| Time-series plots                                                | R14                                 | Defines line plotting of x/y coordinates.                                                                                                     |
| Scatter plots                                                    | R15                                 | Defines scatter plot as paired x/y data positions.                                                                                            |
| Bar plots                                                        | R16                                 | Defines bar plot for categorical comparison by bar height.                                                                                    |
| Ordinary least-squares affine model                              | R03, R04                            | Covers residual-sum-of-squares minimization and degree-1 polynomial fitting.                                                                  |
| Weighted affine/quadratic polynomial fits                        | R04                                 | Covers polynomial least-squares fitting; implementation weights are project-specific inverse-sigma weights.                                   |
| Huber robust affine model and robust weights                     | R06, R12                            | Covers Huber loss/outlier resistance and MAD robust scale.                                                                                    |
| Piecewise-linear monotone model                                  | R05                                 | Covers one-dimensional piecewise linear interpolation and monotonic x-coordinate requirement.                                                 |
| Orthogonal/Deming-style diagnostic affine model                  | R07                                 | Covers ODR as fitting with uncertainty in both input and output variables.                                                                    |
| Hysteresis / reversibility / direction effects                   | R17, R18                            | Covers increasing/decreasing loading, reversibility error, and force-calibration procedures.                                                  |
| Drift, creep, and zero-return diagnostics                        | R17, R18                            | Covers wait periods, creep test, zero after unloading, and drift context.                                                                     |
| RMSE / squared error metrics                                     | R08                                 | RMSE is the square root of MSE as implemented by the report.                                                                                  |
| MAE                                                              | R09                                 | Mean absolute error; zero is best.                                                                                                            |
| R²                                                               | R10                                 | Coefficient of determination; best 1.0; can be negative.                                                                                      |
| Cross-validation metrics                                         | R11                                 | Covers splitting into training/test folds and reporting fold-level performance.                                                               |
| Selection likelihood / normalized decision weights               | R13                                 | Covers softmax normalized exponential weights summing to one.                                                                                 |
| AICc / BIC / adjusted R²                                         | R01, R03, R08, R10                  | Project calculates these from residual sums, sample count, and parameter count. Use as secondary diagnostics with small calibration datasets. |
| Sample statistics: median, standard deviation, sample rate, gaps | R12 plus project implementation     | MAD is externally sourced; medians/std/rates/gaps are straightforward descriptive statistics computed by the implementation.                  |
| Firmware constants / HX711-style scale/offset                    | Project implementation plus R02/R18 | External sources cover calibration curves; exact firmware semantics are project-specific and must be verified against firmware code.          |
