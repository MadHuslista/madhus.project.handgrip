# Contributing to Documentation

## Summary

This folder is reserved for documentation-maintenance rules, style guides, and contribution checklists.

## Current validation path

Use the maintenance docs:

- [`../maintenance/documentation-validation.md`](../maintenance/documentation-validation.md)
- [`../maintenance/handoff-workflow-validation.md`](../maintenance/handoff-workflow-validation.md)

## Minimum contribution rule

Any documentation change that affects firmware schema, stream names, IPC topics, calibration protocol defaults, or config paths must also update:

1. root architecture docs,
2. component docs,
3. validation scripts/checklists,
4. any affected workflow docs.
